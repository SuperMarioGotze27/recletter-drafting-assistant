What It Does

A complete pipeline from student profile to polished recommendation letter

📊

Profile Analysis

Ingests student transcripts, GPA, course highlights, project evidence, and faculty notes to build a comprehensive academic profile.

🎯

Strength Classification

A fine-tuned BERT-tiny model predicts recommendation strength (Moderate / Strong / Exceptional) with confidence scores.

😊

Sentiment Verification

DistilBERT sentiment analysis checks whether faculty notes are sufficiently positive, flagging drafts needing revision.

✍️

LLM Letter Generation

DeepSeek-V3 generates natural, personalized recommendation letters weaving specific evidence into compelling narratives.

✅

Post-hoc Verification

The fine-tuned model re-evaluates the generated letter to ensure stated strength matches predicted classification.

📤

Multi-format Export

Export drafts as TXT, PDF, or DOCX. Batch processing supports generating letters for entire cohorts via CSV upload.
