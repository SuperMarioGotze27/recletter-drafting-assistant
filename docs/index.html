What It Does

A complete pipeline from student profile to polished recommendation letter

📊

Profile Analysis

Ingests student transcripts, GPA, course highlights, project evidence, and faculty notes to build a comprehensive academic profile.

🎯

Strength Classification

A fine-tuned BERT-tiny model predicts recommendation strength (Moderate / Strong / Exceptional) from the student profile with confidence scores.

😊

Sentiment Verification

DistilBERT sentiment analysis checks whether faculty notes are sufficiently positive, flagging drafts that may need revision.

✍️

LLM Letter Generation

OpenAI GPT-4o-mini generates natural, personalized recommendation letters that weave specific evidence into compelling narratives.

✅

Post-hoc Verification

The fine-tuned model re-evaluates the generated letter to ensure the stated recommendation strength matches the predicted classification.

📤

Multi-format Export

Export drafts as TXT, PDF, or DOCX. Batch processing supports generating letters for entire cohorts via CSV upload.

My Contribution

What I built and why it matters

🏗️

End-to-End System Design

Architected the full pipeline from data preprocessing → model fine-tuning → web application → deployment. Made deliberate trade-offs between model size (BERT-tiny for speed) and capability (LLM for generation quality).

🔧

Model Fine-Tuning

Fine-tuned prajjwal1/bert-tiny on synthetic but rule-governed recommendation-strength data. Achieved 100% held-out accuracy, demonstrating the model successfully learned the designed classification rules.

🔗

LLM + ML Hybrid Approach

Integrated OpenAI GPT-4o-mini for natural language generation while using the fine-tuned BERT model for post-hoc verification. This hybrid approach combines LLM fluency with ML model consistency checks.

🚀

Multi-Platform Deployment

Deployed on Streamlit Cloud (primary), Hugging Face Spaces (backup), and GitHub Pages (static showcase). Built auto-ping scripts to prevent free-tier hibernation.
